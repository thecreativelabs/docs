## 2025

### Q2 2025

<Update label="Documentation Portal Launch">
  Launched `help.thecreativelabs.io` — a dedicated documentation and knowledge base for all Creative Labs clients.

  - All 8 documentation categories now available: Getting Started, Shopify Guides, Platform Setup, Integrations, API Reference, FAQs, Troubleshooting
  - Full-text and semantic search across all docs
  - AI-powered "Ask AI" functionality for instant answers
</Update>

<Update label="Support Portal Improvements">
  Upgraded client support experience at `support.thecreativelabs.io`.

  - New support widget with pre-filled ticket categories for faster routing
  - Estimated response times shown per ticket priority
  - Critical ticket escalation path for store-down issues
</Update>

---

### Q1 2025

<Update label="Checkout Extensibility Support">
  All new Shopify Plus projects now built with Checkout Extensibility, replacing the deprecated `checkout.liquid`.

  - Migrated 3 existing Plus clients from legacy checkout
  - Support for custom upsell blocks, trust badges, and gift message fields
</Update>

<Update label="Performance Baseline Update">
  Raised our minimum Lighthouse threshold for all delivered projects.

  - New minimum: **85 on mobile** (up from 75)
  - WebP image conversion now included in all new theme builds by default
</Update>

## 2024

### Q4 2024

<Update label="Shopify Markets Integration">
  Added Shopify Markets support to multi-region client stores.

  - Currency localization and language switching
  - Geo-redirect configuration
</Update>

<Update label="Klaviyo Flow Templates">
  Released 5 pre-built Klaviyo flow templates for new Creative Labs clients.

  - Welcome Series
  - Abandoned Checkout
  - Browse Abandonment
  - Post-Purchase
  - Winback
</Update>

### Q3 2024

<Update label="Recharge Subscriptions Offering">
  Added subscription setup and Recharge integration to the standard service catalog.

  - New internal process for testing subscription flows before launch
  - Recharge customer portal styling included
</Update>

<Update label="Theme Component Library">
  Released an internal reusable Shopify theme component library (Liquid partials + CSS).

  - Reduces average development time by ~20% for standard eCommerce features
  - Consistent, tested patterns across all client builds
</Update>

### Q2 2024

<Update label="Shopify CLI v3 Adoption">
  Migrated all internal tooling to Shopify CLI v3.

  - Faster theme sync
  - Improved error reporting during development
</Update>

### Q1 2024

<Update label="Support SLA Update">
  Improved response time commitments for all support tiers.

  - **Standard tickets:** response within 4 business hours (previously 1 business day)
  - **Critical tickets** (store down, checkout broken): response within 1 hour during business hours
</Update>
